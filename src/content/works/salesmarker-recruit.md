## プロジェクト

採用担当の人事メンバーがあまりウェブサイト作成の経験がない方であったため、イメージを分かりやすく可視化して議論を進めていくことを重視。
まずはサイトマップとサイト内外での動線を整理し、必要なページやコンテンツの整理をしました。

![Sales Marker - Main view](/work/salesmarker-recruit/01.png)

ワイヤーの作成はAIでサイトマップやワイヤーが作成できる<a href="https://www.relume.io/ja">Relume</a>を活用して半日程度で実施。
人事メンバーともやりとりをしながら、効率的にサイトの全体像を決めることができました。

![Sales Marker - Feature 1](/work/salesmarker-recruit/02.png)

ビジュアルデザインを担当したデザイナーと細かくやり取りをしながら、どんなデバイスサイズでもきれいに見られるように実装。
また、応募者には英語話者も多いため、Google翻訳を利用して、英語テキスト作成の工数をかけずに英語に対応するなどの調整を実施しました。

![Sales Marker - Feature 1](/work/salesmarker-recruit/03.png)

## 技術

全体的にモダンな技術への刷新を実施しました。

- タスクランナー
  - Gulp → Vite
- スタイリング
  - Sass → Tailwind.css
- WordPressのカスタムフィールド
  - CustomFieldSuite → SecureCustomField

またFigmaMCPを活用して、デザインデータから実装の叩き台を作成するなどの効率化にも取り組みました。
